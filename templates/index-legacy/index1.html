Why Knowledge Garden?

Built for developers who work across multiple machines

Works Anywhere

SSH into a remote server and capture notes. Your laptop, work machine, and servers all share the same knowledge base. No GUI required.

Full-Text Search

PostgreSQL-powered full-text search finds notes instantly across thousands of documents. Type tolerance handles mistakes automatically.

Wiki-Style Links

Write [[Note Title]] to create automatic bidirectional links. Build a knowledge graph that grows organically over time.
