Features

Works Anywhere

SSH into remote servers and capture notes. Your knowledge base follows you everywhere—no GUI required.

$ ssh server@prod
$ kg note create --title "Bug fix"
func search(query)

Full-Text Search

PostgreSQL FTS with tsvector. Type tolerance, ranking, instant results across thousands of notes.

$ kg search "golang"
→ Found 23 notes in 12ms
