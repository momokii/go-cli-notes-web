üîê Authentication Commands

kg-cli register Create a new account
+

Create a new Knowledge Garden account with interactive prompts.

kg-cli register
Interactive prompts:
  • Username: Your username (3-100 characters, alphanumeric only)
  • Email: Your email address
  • Password: Your password (minimum 8 characters)
kg-cli login Authenticate with credentials
+

Log in to your Knowledge Garden account.

kg-cli login
Prompts for:
  • Email address
  • Password
kg-cli logout Clear stored credentials
+

Log out and clear your stored authentication token.

kg-cli logout
kg-cli status Check authentication status
+

Display your current authentication status and API connection details.

kg-cli status

==========================
API URL: http://localhost:8080
Status: Authenticated
Email: user@example.com

üìù Note Commands

kg-cli note list Display all notes
+
kg-cli note list [flags]
Flag Short Description Default
--page-pPage number1
--limit-lNotes per page (1-100)20
--search-sSearch query-
--tag-tFilter by tag name/ID-
Examples:
kg-cli note list
kg-cli note list --page 2 --limit 50
kg-cli note list --search "golang" --tag "programming"
kg-cli note get <id> Display a specific note
+
kg-cli note get <note-id>

Displays the full content and metadata of a note by its UUID.

kg-cli note create Create a new note
+
kg-cli note create [flags]
Flag Short Description Default
--title-tNote title (required)-
--content-cNote contentEmpty
--type-TNote typenote
Note types: note, daily, meeting, idea
kg-cli note create -t "My Note" -c "Content here"
kg-cli note create --title "Meeting" --type meeting
kg-cli note search Full-text search
+
kg-cli note search <query> [flags]

Search notes using PostgreSQL full-text search with fuzzy matching.

kg-cli note search "golang"
kg-cli note search "full-text search" --limit 10
kg-cli note daily Get/create daily note
+
kg-cli note daily [date]

Get or create a daily note. Date format: YYYY-MM-DD or "today" (default).

kg-cli note daily
kg-cli note daily 2026-01-04

üîó Wiki-Style Links

üîë Key Concept

Connect notes automatically using [[Note Title]] syntax. Links are created when you save a note that references existing notes.

Syntax

This note is related to [[Another Note]].

Multiple links: [[Note 1]], [[Note 2]], and [[Note 3]].

Spaces work: [[My Long Note Title]].

How It Works

1
Create the target note first (e.g., "Go Programming Tips")
2
Create/update another note with [[Go Programming Tips]] in content
3
Link is created automatically! Use kg-cli note links to verify

‚ö†Ô∏è Important: Target note must exist before creating the link. Use exact title (case-insensitive).
